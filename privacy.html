Политика конфиденциальности OkTolk

Дата последнего обновления: 11 мая 2026 г.

Настоящая Политика конфиденциальности описывает, как Индивидуальный предприниматель Коваленко Евгений Сергеевич (далее — «Оператор», «мы») обрабатывает персональные данные пользователей приложения OkTolk (oktolk.ru).

1. Какие данные мы собираем

  • Имя и номер телефона — при регистрации в приложении
  • Текст обращений в AI-чат — для формирования ответа
  • Технические данные — тип устройства, браузер, IP-адрес (в целях безопасности)
  • Данные о платежах — обрабатываются платёжным сервисом ЮKassa, мы не храним реквизиты карт

2. Как мы используем данные

  • Для предоставления функций приложения (чат, новости, антискам)
  • Для отправки уведомлений (с вашего согласия)
  • Для улучшения качества сервиса
  • Мы не продаём и не передаём данные третьим лицам в коммерческих целях

3. Хранение данных

Данные хранятся на серверах в России (Amvera Cloud). Текст обращений в AI-чат передаётся в API сторонних AI-провайдеров (AItunnel) исключительно для формирования ответа и не хранится ими.

4. Ваши права

Вы вправе в любое время запросить, исправить или удалить свои персональные данные. Для этого напишите нам: info@oktolk.ru

5. Cookies

Приложение использует локальное хранилище браузера (localStorage) для сохранения настроек и данных профиля. Данные хранятся только на вашем устройстве.

6. Согласие

Используя приложение OkTolk, вы соглашаетесь с настоящей Политикой конфиденциальности. Если вы не согласны — пожалуйста, прекратите использование приложения.
